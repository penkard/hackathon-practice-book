# Chapter 6: Troubleshooting

Every project breaks.  
Real developers fix stuff, not avoid it.

Here’s what to document:
- Common beginner mistakes  
- Error messages you’ll likely see  
- How to debug step-by-step  
- What logs to check  
- What tools help you verify setups  

---

## Practice: Trigger an Error  
Yes, intentionally.

Try:
- Wrong config  
- Missing file  
- Invalid input  
- Broken step  

Then fix it.

:::tip Debugging Trick  
Ask your chatbot questions like:  
"What would cause this to fail?"  
"Which part of the flow should I check first?"  
:::
