# Chapter 8: Architecture

Here’s the simple architecture of the entire project.

- **Docusaurus**: Book platform  
- **FastAPI**: Backend brain  
- **Qdrant**: Vector memory  
- **ChatKit / LLM**: Chat generation  
- **GitHub Pages**: Hosting
